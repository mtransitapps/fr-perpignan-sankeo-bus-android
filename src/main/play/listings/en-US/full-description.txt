This application adds Perpignan Sankéo buses information to MonTransit.

This app contains the buses schedule and the latest news from www.sankeo.com and @SANKEOtrafic on Twitter.

Sankéo serves Perpignan in Pyrénées-Orientales, France.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link https://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by Sankéo.
https://transport.data.gouv.fr/datasets/gtfs-sankeo?locale=en

This application is free and open-source:
https://github.com/mtransitapps/fr-perpignan-sankeo-bus-android

This app is not related with Sankéo, Perpignan Méditerranée Métropole & Keolis Perpignan Méditerranée.

Permissions:
- Other: required to read news from www.sankeo.com and Twitter.
